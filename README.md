# 流氓软件数据库

本数据库采用 **ini** 文本格式标识每一个在列的流氓软件的信息。

格式如下：

**[软件名]<br/>
URL = 软件网址<br/>
DLURL = 下载地址<br/>
Feature = 主要特征<br/>
Dev = 软件公司<br/>
CertFP = 证书指纹<br/>
InstDir_x86 = "x86安装路径"<br/>
InstDir_x64 = "x64安装路径"**
